Войти в IT

Доброго времени суток!!! Меня зовут Константин. Мне 42 года и я учусь на Fullstack разработчик на Python. Интересоваться программрованием начал в конце 5 курса военного института, когда появилась необходимость создать программу для защиты дипломного проекта. Тогда это был простой код на паскале для математических расчета. После окончания института прошло несколько лет когда я не был ни как связан с IT. Но по прошествию нескольких лет перейдя на инженерную должность пришлось заниматься администрирование сетей подразделения. И вот сбылась мечта я учусь на программиста!

Цель

Получить должность Junior Fullstack разработчика в компании, где я могу применять свои навыки и знания в разработке на Python, специализируясь на Django, DRF, Telegram Bot API и AioGram.

Профессиональный опыт

  1. Пет-проект "Доска объявлений" (Python, Django):

    • Разработал полностью функционирующую веб-платформу для публикации объявлений, где пользователи могут создавать учетные записи, размещать, редактировать и удалять объявления.
    • Реализовал функционал поиска, фильтрации и сортировки объявлений.
    • Использовал Django ORM для работы с базой данных и шаблонизатор Django для создания пользовательского интерфейса.

  2. Пет-проект "Новостной портал" (Python, Django, DRF):

    • Создал RESTful API для получения и отображения новостных статей.
    • Разработал систему аутентификации и авторизации пользователей с помощью Django REST Framework.
    • Использовал DRF Serializers для преобразования данных в формат JSON.

  3. Пет-проект "Telegram бот" (Python, Telegram Bot API):

    • Написал Telegram-бота для автоматической рассылки новостей из новостного портала на основе подписки пользователей.
    • Настроил обработку команд и взаимодействие с API Telegram для получения и отправки сообщений.
    • Использовал библиотеку python-telegram-bot для упрощения работы с Telegram API.

  4. Пет-проект "AioGram" (Python, AioGram):

    • Использовал AioGram, асинхронную библиотеку для работы с Telegram Bot API, для создания Telegram-бота с асинхронной обработкой запросов.
    • Реализовал функции, такие как отправка сообщений, ответ на команды и обработка обновлений с использованием асинхронных методов.
